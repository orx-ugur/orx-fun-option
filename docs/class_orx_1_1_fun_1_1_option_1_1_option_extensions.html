Flattens the option of option of T . Maps Opt<Opt<T>> to Opt<T> as follows:

  • None => None<T>(),
  • Some(None<T>()) => None<T>(),
  • Some(Some(T)) => Some(T).

Assert(None<Opt<char>>().Flatten() == None<char>()); Assert(Some(None<char>()).Flatten() == None<char>()); Assert(Some(Some('c')).Flatten() == Some('c'));

Allows an option of a tuple (t1, t2) to map with a function taking two arguments t1 and t2.

static int Add(int a, int b) => a + b;

var numbers = Some((1, 2)); var sum = numbers.Map(Add); Assert(sum == Some(3));

This is mostly useful in enabling function composition.

Creates a result of T as Some variant with value value if the someCondition holds. Otherwise, it will return the None variant.

string team = "secret"; int score = 42;

Opt<string> winner = SomeIf(score > 30, team); Assert(winner == Some(team));

Opt<string> loser = SomeIf(score < 40, team); Assert(loser.IsNone);

Creates an option of T as Some variant with the given value . However, if the value is null, it will map into None.

optName = GetName(42).SomeIfNotnull(); Assert.True(optName.IsNone);
