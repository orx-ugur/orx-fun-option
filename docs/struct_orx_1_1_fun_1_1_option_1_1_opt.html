Returns back None if IsNone. Otherwise, returns Some(value) if validationCriterion (value) holds; None if it does not hold. Especially useful in fluent input validation.

static Opt<Account> MaybeParseAccount(..) { } static bool IsAccountNumberValid(int number) { } static bool DoesAccountExist(string code) { }

var account = MaybeParseAccount(..) .Validate(acc => IsAccountNumberValid(acc.Number)) .Validate(acc => DoesAccountExist(acc.Code)); // account will be Some(account) only if: // - MaybeParseAccount returns Some(account), and further, // - both IsAccountNumberValid and DoesAccountExist validation checks return true.
